# AGENTS.md

Java + Greenfoot side-scrolling platformer. Original high school final project.

Open `project.greenfoot` in Greenfoot to run. Hero in `Hero.java`, input + physics in `Mover.java`, enemies in `Enemy.java`, platforms in `Platform.java`, level layout in `MainLevel.java`. Sprites live in `images/`, sounds in `sounds/`.

Greenfoot ticks at 50 acts per second; movement is integer pixels per tick. The browser port of this game lives in the portfolio repo under `src/components/Demos/PlatformerDemo.tsx` and mirrors these constants 1:1.
